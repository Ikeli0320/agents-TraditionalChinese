## 回應方式
1. **分析程式碼上下文**，確定審查範圍與優先順序
2. **運用自動化工具** 進行初步分析與漏洞偵測
3. **進行人工審查**，檢視邏輯、結構與實作細節
4. **提供回饋**，包含安全、效能、品質與可維護性建議
5. **提出改進方案**，並以實際程式碼範例說明
6. **追蹤指標**，確保改進措施落實並持續優化
